# App Icons

This directory should contain the following icon files for proper app packaging:

- `icon.png` - Main app icon (512x512 PNG)
- `icon.ico` - Windows icon file
- `icon.icns` - macOS icon file

## Icon Requirements:

### Windows (icon.ico)
- Multi-resolution ICO file containing: 16x16, 24x24, 32x32, 48x48, 64x64, 128x128, 256x256

### macOS (icon.icns)
- ICNS file containing multiple resolutions from 16x16 to 1024x1024

### Linux (icon.png)
- 512x512 PNG file

## Creating Icons:

You can use tools like:
- **Online**: favicon.io, icoconvert.com
- **macOS**: Icon Composer, IconFly
- **Cross-platform**: GIMP, ImageMagick
